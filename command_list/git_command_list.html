Gitコマンド一覧

ローカルリポジトリの作成

コマンド：git init
initは「initializeの略である

Gitリポジトリのコピーを作成する

コマンド：git clone ＜リポジトリ名＞
クローン（コピー）を作成する。＜リポジトリ名＞のところには「https://github.com/itpracticeaddress/IT_Material」などを入力する

変更をステージに追加する

コマンド：git add ＜ファイル名、ディレクトリ名など＞
git add . は変更分をすべてステージに追加するということである

変更を記録する（コミット）

コマンド：git commit
editorを登録しているとeditor画面が開く
git commit - m "コミットメッセージ" でメッセージ付きで変更を記録する

現在の変更状況を確認する

コマンド：git status
変更されたファイルを確認する

変更差分を確認する

コマンド：git diff
git addする前の変更分についてはgit diff やgit diff ＜ファイル名＞を使う
git add した後の変更分については git add --stagedを使う
diffはdifferenceの略である

変更履歴を確認する

コマンド：git log
一行で表示するときはgit log --oneline
ファイルの変更差分を表示するときはgit log -p ＜ファイル名＞
表示するコミット数を制限するときはgit log -n ＜コミット数＞

ファイルの削除を記録する

コマンド：git rm ＜ファイル名＞
git rm -r ＜ディレクトリ名＞
ファイルを残したいときはgit rm --cached ＜ファイル名＞
